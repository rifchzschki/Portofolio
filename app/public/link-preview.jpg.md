# Link Preview Image Instructions

Anda perlu membuat file `link-preview.jpg` di folder ini untuk social media preview.

## Spesifikasi:

- **Ukuran:** 1200x630 pixels
- **Format:** JPG atau PNG
- **Konten:** Foto/design yang mewakili portfolio Anda

## Cara Membuat:

### Opsi 1: Menggunakan Canva (Gratis & Mudah)

1. Buka https://www.canva.com
2. Pilih template "Facebook Post" atau "LinkedIn Post"
3. Atur ukuran custom: 1200 x 630 px
4. Desain dengan:
   - Nama Anda: "Rifki Virzya"
   - Tagline: "Full-Stack Developer & Software Engineer"
   - Background warna yang sesuai dengan brand Anda
   - Foto profil (opsional)
5. Download sebagai JPG
6. Rename menjadi `link-preview.jpg`
7. Taruh di folder `public/`

### Opsi 2: Menggunakan Tools Online

- https://www.opengraph.xyz/
- https://metatags.io/
- https://www.bannerbear.com/

### Opsi 3: Placeholder Sementara

Jika belum siap, Anda bisa menggunakan salah satu icon yang sudah ada:

- Rename `apple-icon.png` menjadi `link-preview.jpg`
- Atau buat solid color image dengan text

## Setelah Selesai:

Hapus file ini (`link-preview.jpg.md`) dan pastikan file `link-preview.jpg` ada di folder `public/`.

## Testing:

Setelah deploy, test preview Anda di:

- https://www.opengraph.xyz/
- https://cards-dev.twitter.com/validator
- https://www.linkedin.com/post-inspector/
